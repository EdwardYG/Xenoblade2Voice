# 异度神剑2战斗后语音查看程序

### Introduction

根据选择的人物筛选并播放相关角色的战斗后语音

### Feature

1. 切换中文/日文
2. 点击角色头像选择最多三个角色
3. 筛选选择角色的相关对话组
4. 播放对话组内的所有对话并显示相应的文本
5. 双击文本可以播放该文本的语音
